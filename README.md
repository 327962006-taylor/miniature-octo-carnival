# SOUND ARCHITECT

赛博朋克风格的音乐/声音设计师个人品牌网站

## 快速开始

1. 克隆仓库
2. 将你的音频文件放入 `assets/` 目录，命名为 `track1.mp3`, `track2.mp3`, `track3.mp3`
3. 编辑 `index.html` 中的个人信息
4. 推送到 GitHub，启用 GitHub Pages

## 部署

网站通过 GitHub Pages 自动部署。推送到 `main` 分支后，网站将在 `https://[username].github.io/sound-portfolio/` 可用。

## 自定义域名（可选）

在 `docs/CNAME` 文件中添加你的域名，详情见 [GitHub Pages 文档](https://docs.github.com/en/pages/configuring-a-custom-domain-for-your-github-pages-site)。
